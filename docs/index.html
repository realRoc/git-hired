How To Start

怎么开始测试

Start from the one-line command above. It works with Claude Code, Codex, Notion AI, or any work agent with knowledge-base and memory capability. The agent should treat `skill.md` as active instructions, ask your target role first, fall back to your current profession or identity when needed, keep the default mode as `history-only`, and touch only the projects or files you explicitly authorize. We never upload your local repo or file data to our server.
从上面这一行命令开始。它支持 Claude Code、Codex、Notion AI，或任何具备知识库和记忆能力的工作 agent。agent 会把 `skill.md` 当作当前会话指令，先问你的目标岗位；如果暂时没有，再回到你当前的职业或身份；默认保持 `history-only`，只会访问你明确授权的项目或文件，也不会把你的本地 repo 或文件数据上传到我们的服务器。
  • Paste the one-line `read .../skill.md` command into your own work agent, such as Claude Code, Codex, Notion AI, or another memory-enabled agent.
  • Answer the role-routing and privacy questions directly inside that agent session.
  • Keep the short `HIRED` snapshot, and optionally attach the markdown report if you decide to apply.
  • 把一行 `read .../skill.md` 命令粘贴到你自己的工作 agent 里运行，例如 Claude Code、Codex、Notion AI，或其他具备知识库和记忆能力的 agent。
  • 直接在那个 agent 会话里回答岗位路由和隐私边界问题。
  • 保留终端里的 `HIRED` 快照；如果后续要申请，也可以附上本地生成的 markdown 详细报告。

What You'll Get

跑完之后你会拿到什么

  • A clearer view of how you actually use AI, not just how you would describe your workflow.
  • A shareable `HIRED` snapshot with your MBTI work personality, ability score, compact signal board, and matching ASCII card.
  • A local markdown report you can keep for yourself, use for reflection, or attach later if you decide to apply.
  • 你真实的 AI 使用方式，而不只是你会如何描述自己的工作流。
  • 一份可传播的 `HIRED` 工作画像，包含你的 MBTI 工作人格、能力值、紧凑信号板和匹配的 ASCII 卡片。
  • 一份本地 markdown 详细报告，你可以自己留存、复盘，或在之后决定申请时附上。
