How To Send This

怎么发给候选人

Send the relevant role page to the candidate. They paste the prompt into their own Claude Code or Codex, run it locally, and send the resulting report back to you.
把岗位对应页面发给候选人，让他把页面里的 prompt 粘贴到自己的 Claude Code 或 Codex 中执行，再把结果发回给你。
  • Useful for screening AI-native roles or adding signal before an interview.
  • This is a hiring input, not a monitoring tool.
  • The prompts ask for aggregate evidence, redacted examples, and structured conclusions only.
  • 适合 AI Native 岗位的初筛，或在面试前补充行为信号。
  • 这是招聘输入，不是监控工具。
  • prompt 只要求宏观分布、脱敏例子和结构化结论。

What You Get

你会得到什么

  • How the candidate actually uses AI, not how they say they use AI.
  • A clearer sense of whether this person is a builder, PM, operator, or just fluent in buzzwords.
  • A list of interview questions tied to observed behavior.
  • 候选人真实的 AI 使用习惯，而不是他对自己使用习惯的描述。
  • 更快判断这个人更像 builder、PM、operator，还是只是会说术语。
  • 一组可以直接带进面试的、基于证据的追问问题。
