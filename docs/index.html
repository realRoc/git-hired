怎么发给候选人

把对应岗位页面发给候选人，让他把页面里的 prompt 完整粘贴到自己的 Claude Code 或 Codex 中执行，再把结果发回给你。
  • 适合用在 AI Native 岗位的初筛或加深判断。
  • 输出面向招聘判断，不是监控工具。
  • 只要求宏观分布、去标识化证据和结构化结论。

你会得到什么

  • 候选人的 AI 使用习惯，而不是他对自己使用习惯的描述。
  • 这个人更像 builder、PM、operator，还是只是会说术语。
  • 一个可直接带入面试的追问清单。
