`fatal: not a qualified candidate`。`git-hired` 是一个从真实工作方式出发，测试你 MBTI 工作人格的工具。让你自己的工作 agent 去分析你到底是怎么工作的，无论它是 Claude Code、Codex、Notion AI，还是其他具备知识库和记忆能力的工作 agent。

Privacy first: we never upload your local repo, files, or any other local data to our server.

What You Get

你会拿到什么

Works with Claude Code, Codex, Notion AI, or any work agent with knowledge-base and memory capability. If the runtime is a rich-text surface, the result should fall back to a compact card instead of wide terminal ASCII.
支持 Claude Code、Codex、Notion AI，或任何具备知识库和记忆能力的工作 agent。如果运行环境是富文本界面，结果应自动退化成紧凑卡片，而不是宽终端 ASCII。
  • A clearer view of how you actually use AI, not just how you would describe your workflow.
  • A shareable `HIRED` snapshot with your MBTI work personality, ability score, compact signal board, and matching ASCII card.
  • A local markdown report you can keep for yourself, use for reflection, or attach later if you decide to apply.
  • 你真实的 AI 使用方式，而不只是你会如何描述自己的工作流。
  • 一份可传播的 `HIRED` 工作画像，包含你的 MBTI 工作人格、能力值、紧凑信号板和匹配的 ASCII 卡片。
  • 一份本地 markdown 详细报告，你可以自己留存、复盘，或在之后决定申请时附上。
