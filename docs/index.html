git hired

Hiring for AI-native teams is broken. Resumes show claims. Interviews show performance. Take-home tests show isolated skill. git-hired asks a different question: can this person use agents to understand ambiguity, inspect real work traces, make tradeoffs, and ship?

AI-native 团队的招人方式坏掉了。简历展示的是声明，面试展示的是表演，作业题展示的是孤立技能。 git-hired 问的是另一个问题：这个人能不能用 agent 理解模糊任务、检查真实工作痕迹、做取舍，并把事情推进到交付？

Open-source AI-native hiring protocol. Candidates run it inside their own work agent; `git-hired` never uploads local repo, file, or local data to our server.

开源 AI-native 招人协议。候选人在自己的工作 agent 里运行；`git-hired` 不会把本地 repo、文件或本地数据上传到我们的服务器。

03 /

what you get

你会拿到什么

runtime
Works with Claude Code, Codex, Notion AI, or any work agent with knowledge-base and memory. On rich-text surfaces the result falls back to a compact card instead of wide terminal ASCII.
支持 Claude Code、Codex、Notion AI，或任何具备知识库和记忆能力的工作 agent。富文本界面下结果会自动退化成紧凑卡片，而不是宽终端 ASCII。
signal
Work-trace evidence, actual agent workflow, role-specific fit, and MBTI work personality as a bounded work-style read.
工作痕迹证据、真实 agent 工作流、角色匹配度，以及有边界的 MBTI 工作人格判断。
report
A privacy-bounded fit report with evidence confidence, strengths, risks, next steps, and a candidate-facing local markdown report.
一份带隐私边界的匹配报告：证据置信度、优势、风险、下一步，以及候选人可留存的本地 markdown 详细报告。
