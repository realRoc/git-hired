How To Start

怎么开始测试

Pick the closest role page, or start from the Universal Entry if you are changing tracks or not sure which role fits best. The prompt works with Claude Code, Codex, Notion AI, or any work agent with knowledge-base and memory capability. By default it stays history-only, only touches projects or files you explicitly authorize, and never uploads your local repo or file data to our server.
选择最接近你的岗位页；如果你正在转岗，或暂时不确定该选哪个岗位，就从通用测试入口开始。这个 prompt 支持 Claude Code、Codex、Notion AI，或任何具备知识库和记忆能力的工作 agent。默认只看历史记录，只会访问你明确授权的项目或文件，也不会把你的本地 repo 或文件数据上传到我们的服务器。
  • Pick the test that best matches what you do now or what you want to do next.
  • Run the prompt inside your own work agent, such as Claude Code, Codex, Notion AI, or another memory-enabled agent.
  • Keep the short `HIRED` snapshot, and optionally attach the markdown report if you decide to apply.
  • 选择最接近你当前职业，或你下一步目标职业的测试入口。
  • 在你自己的工作 agent 里运行 prompt，例如 Claude Code、Codex、Notion AI，或其他具备知识库和记忆能力的 agent。
  • 保留终端里的 `HIRED` 快照；如果后续要申请，也可以附上本地生成的 markdown 详细报告。

What You'll Get

跑完之后你会拿到什么

  • A clearer view of how you actually use AI, not just how you would describe your workflow.
  • A shareable `HIRED` snapshot with your MBTI work personality, ability score, compact signal board, and matching ASCII card.
  • A local markdown report you can keep for yourself, use for reflection, or attach later if you decide to apply.
  • 你真实的 AI 使用方式，而不只是你会如何描述自己的工作流。
  • 一份可传播的 `HIRED` 工作画像，包含你的 MBTI 工作人格、能力值、紧凑信号板和匹配的 ASCII 卡片。
  • 一份本地 markdown 详细报告，你可以自己留存、复盘，或在之后决定申请时附上。
