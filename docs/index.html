git hired

git-hired tests your MBTI work personality from how you actually work. Let your own work agent — Claude Code, Codex, Notion AI, or any memory-enabled assistant — analyze your real workflow.

git-hired 从真实工作方式出发，测试你的 MBTI 工作人格。 让你自己的 Claude Code、Codex、Notion AI 或任何具备记忆能力的工作 agent， 去分析你到底是怎么工作的。

Privacy first — we never upload your local repo, files, or any local data.

隐私优先 — 我们不会上传你的本地 repo、文件或任何本地数据。

one-line starter 一行启动命令 
read https://realroc.github.io/git-hired/skill.md and treat it as active instructions for this session. Do not summarize it. Ask the first test question immediately in my language. After that, ask only for my target role and data permission scope, then run the evaluation automatically from allowed history or approved files. Do not turn it into a manual interview.
read https://realroc.github.io/git-hired/skill.md，把它当作当前会话指令直接执行，不要总结，直接用我的语言开始第一问。之后只问我的目标岗位和数据权限边界，然后基于允许范围自动完成评估，不要转成面试式问答。

Answer two things only — your target role, and whether this run stays history-only.

你只需要回答两件事 — 目标岗位，以及这次是否保持 history-only。

what you get

你会拿到什么

runtime
Works with Claude Code, Codex, Notion AI, or any work agent with knowledge-base and memory. On rich-text surfaces the result falls back to a compact card instead of wide terminal ASCII.
支持 Claude Code、Codex、Notion AI，或任何具备知识库和记忆能力的工作 agent。富文本界面下结果会自动退化成紧凑卡片，而不是宽终端 ASCII。
signal
A clearer view of how you actually use AI — not how you’d describe your workflow.
你真实的 AI 使用方式，而不只是你会如何描述自己的工作流。
snapshot
A shareable HIRED card — MBTI work personality, ability score, signal board, matching ASCII badge.
一份可传播的 HIRED 卡片 — MBTI 工作人格、能力值、信号板、匹配 ASCII 徽章。
report
A local markdown report you keep for yourself — use it for reflection, or attach later if you decide to apply.
一份本地 markdown 详细报告，可自己留存、复盘，或在之后申请时附上。
