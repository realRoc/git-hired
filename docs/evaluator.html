How To Evaluate

如何评估

confidence
Check whether the GH level is backed by real traces, public next-step evidence, a deeper report, or only self-report.
先看 GH Level 背后是真实痕迹、公开 next-step evidence、更深入报告，还是只有自我描述。
workflow
Look for agent orchestration, spec refinement, debugging maturity, and outcome judgment.
看 agent 编排、spec 压缩、调试成熟度和结果判断。
trust
Respect privacy scope. Do not ask for raw local files, private transcripts, or secret-bearing evidence in public. Use the public-safe card first.
尊重隐私范围。不要在公开场景要求原始本地文件、私人 transcript 或带 secret 的证据。先看可公开卡片。
decision
Decide strong yes / trial / pass from the market level card, evidence report when shared, rubric, and any follow-up task.
基于 market level 卡片、对方愿意分享的证据报告、rubric 和后续真实任务，决定 strong yes / trial / pass。
