How To Evaluate

如何评估

confidence
Check whether the report is based on real traces, thin history-only evidence, or self-report.
先看报告基于真实痕迹、薄弱 history-only 证据，还是自我描述。
workflow
Look for agent orchestration, spec refinement, debugging maturity, and outcome judgment.
看 agent 编排、spec 压缩、调试成熟度和结果判断。
trust
Respect privacy scope. Do not ask for raw local files, private transcripts, or secret-bearing evidence in public.
尊重隐私范围。不要在公开场景要求原始本地文件、私人 transcript 或带 secret 的证据。
decision
Decide strong yes / trial / pass from the report, rubric, and any follow-up task.
基于报告、rubric 和后续真实任务，决定 strong yes / trial / pass。
